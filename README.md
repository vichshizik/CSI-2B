CSI-2B is a program for solving radial Schrodinger equation using cubic spline interpolation method. Solver for two-body radial Schrodinger equation using spline-interpolation method. The program built with Qt6 library with GUI.
The CSI method is implemented as standalone program written on C++ language using [Qt 6 library](https://www.qt.io/). CSI-2B uses [ExprTk library](https://www.partow.net/programming/exprtk/) for parsing the potential function, [QCustomPlot library](https://www.qcustomplot.com/) for graph display and [Eigen library](https://libeigen.gitlab.io/) for matrix computations.
You can check out the documentation [here](./Program_desciption.md).

Installing
There are two ways to get the program.
1. Download pre-compiled version for platform on [release page](https://github.com/vichshizik/CSI-2B/releases)
2. Compile the program using the provided source code in the repository.
